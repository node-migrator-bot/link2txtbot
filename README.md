# link2txtbot

## Install & Run

```bash
sudo apt-get install caca-utils #You need the python "img2txt" util in your path
git clone git@github.com:jesusabdullah/link2txtbot.git
cd link2txtbot
npm install
# You will want to modify link2txtbot to "point" to your own desired channel.
node ./link2txtbot.js
```
